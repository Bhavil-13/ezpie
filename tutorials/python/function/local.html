Local and non-local variables

In this article you will get to know more about your variables you use.

What's a local variable?

When you create a function all the variables in it are called local variables.
These variables are only accessible within the function and can not be therefore used outside the function.

Output:

This is a local variable
NameError: name 'variable' is not defined

That gives an error as variable 'variable' is inside function do() so python thinks both the variables are different.

What's a non-local variable?

Non-local may sound like the variable is accessible only outside a function right?
But no! non-local are those which are inside a nested function(Function inside a function)

Means That non-local variables can change local variables!
Non-local variables are created using the nonlocal keyword

Take an example

Here we have a function called outer() which is the main function basically. Next we have a variable outer
with value 'local' as it's a local variable. Next we have another function inner(), which is having a variable outer.
But it's been having the keyword nonlocal. Means we have converted the variable outer into a nonlocal variable.

Any changes made to nonlocal variable outer will change the local variable outer.
