Lambda the anonymous function

In This article you will get to know about one of the short form of a function in python

What is a Lambda function?

Lambda is a tiny small anonymous function which can take any number of arguments, but only one expression

The normal functions are defined using the def keyword in Python, and anonymous functions are defined
using the lambda keyword.

How to define a lambda function?

To define a lambda function we just need a variable! Yes a variable, which is going to take the keyword lambda
and with it a parameter, after which we will add a colon the an expression.

Syntax

a_variable = lambda arguments: expression

Example:

sum = lambda a, b: a+b
sum(12, 42) # sum will be: 54

This is how it would look if you use the normal function

Why do we use the lambda function over normal function?

Lambda functions are used when we require a nameless function for a short period of time.
Mainly it is used as an argument for a normal function. For example the map() function.
