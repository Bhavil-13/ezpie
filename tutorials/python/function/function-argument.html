Arguments in a function

In this article you will learn what and how to use an argument in a function

What is an argument?

In a function most of the time we need to pass some information which can be passed as parameters.

Arguments are give to a function after the function name inside the parentheses.we can add as many
arguments as we want, by separating them with a comma.

Parameter or Argument?

You may have got a bet confuse of parameter or argument right? will no more now,
a parameter is the one inside the round brackets and argument is the value passed to the parameter.
But infact both are the same!

Numbers of Parameters

A function must be called with the same number of argument as the same number of parameters.
Means: 2 parameters so 2 arguments.

Arbitrary for many arguments

Imagine if you need to take 10 arguments! it would be a miss of parameters right? Will what to do then?
Sample use the * sign before a parameter.

This way the function will be able to take a lot of arguments an one parameter!
This would infact make the parameter in a tuple!

Keyword Arguments

we can also send arguments with the key = value style. This way the order of the arguments does not even matter!

Arbitrary Keyword in a function

If we have to add many keyword arguments in a function, just add two asterisk: ** before the parameter name.

This way that function will receive a dictionary of arguments and we can access the items using indexing
