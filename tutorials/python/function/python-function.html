Functions in python

In this article you will get to know about functions

What is function?

A function is like an if statement it will run only when it is called. It's a block of code that can be used repeatedly.

Functions help us by breaking down our program into smaller block of code, as a program gets bigger and larger
in size, functions are very useful.
And since a function runs when it's called it also stops the repeation of code and makes it reusable.

How to create a function

To create a function we need to follow some rules, which are:

  1. A function must start with a def keyword
  2. A function must have a name. Example.. function_name
  3. A function must have a round brackets ()
  4. A function may or may not have some parameters. Example.. def function_name(parameter)
  5. A function when defined in the first line it must end with colon : Example.. def function_name():
  6. A function can have a docstring which is used to describe what a function does
    docstring are written inside triple quotes """ """
  7. A function may or may not have a return statement.
    But it is strongly recommended to use return statement

How to call a function

To call a function we just need to type the name of the function with parameters if given.
That means if I define a function called name() then I can called it by typing name()

Here we have define a function called name() which has two variable name
and text, with name having my name and text using the format() method to format a text. Then just print it.
Next we are simply calling the function by typing its name, that's because we are just going to print the value
not store it in a variable. Since we are not using the return statement.

Docstring in a function

Docstring just tells us what the function does.
May sound waste of time but it is a good programming practice, so adding it is not bad nor good.
If you can remember what that function does!

Docstring are mainly written in triple quotes so that they can extend up to multiple lines.
we can also access what the docstring is using *.__doc__.
But if you are using a code editor like vscode then if you hover over the function called, you can see the docstring.
But back to using *.__doc__

>>> print(name.__doc__)
function to print some format text
