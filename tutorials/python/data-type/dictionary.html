Dictionary in python

In this article you will learn about a very useful data type called dictionary.

A dictionary in python is used for storing data in a way of key and value like key:value
A dictionary is an ordered, changeable and no to duplicate collection.

Note: as of python 3.7 dictionary are ordered, but python 3.6 or older are not ordered!

How to create a dictionary in python

In python a dictionary is written with curly brackets and have keys and values.

A dictionary must have a key with a value like this key:value, and with multiply keys they must be separated by a comma.

Values of a key can be of any data type. BUT the key can only be of string or integer and must be unique.

a = {
'company': 'apple',
'CEO': 'Tim cook',
'Co-founder': 'Steve Jobs',
}

How to access an item on a dictionary

To access an item on a dictionary we use the get() method or simple indexing.

It may not matter which to use but you must know what the difference are any way.
The get() method when used to access any key which does not even exist then it just prints None.
But if you used the indexing way(means the square brackets [ ]) then it will raise an error called KeyError.

How to add and change item on a dictionary

To add a item or change the value of existing item we can use the assignment operator

But if a key mentioned does not exist then a new key with the value will be created.

How to remove an item from a dictionary

Like in any other sequence data type, we can also remove items from a dictionary
to do this we use the pop() and popitem() methods.

The only difference between these two is that pop() is used to remove a value given
while popitem() is used to remove the last item on a dictionary

How to check for keys in python?

In python we can check if a given key is there in the dictionary or not, by using the in or
by combining the not and in keyword like this: not in

a_dict = {'a': 'a', 'b' : 'b', 'c': 'c' , 'd': 'd'}

print('a' in a_dict)
print('b' not in a_dict)
print('c' in a_dict)
