Sets in python

In this article you will learn about one of the collection data type set.

What is a set?

Set is an unchangeable, unordered and unindexed data type and one of the collection data type

Remember: set is not a Sequence data type it is a diffrent data type(set type)

Sets are written inside curly brackets

set = {12, 43, 15}

Rules while making set

Yes there are rules to be followed while making sets.
Here are examples:

# sets should not have duplicate items
a_set = {1, 2, 2, 3, 4, 4} # output: {1, 2, 3, 4}

# sets are not mutable means another collection item is not allowed
# [32, 42] is a list therefore it will give an error
b_set = {12, [32, 42 ]}

Modifying sets

Because set are unordered and unindexed they can not be access using index.

If you don't know the meaning of index it does not matter for now.

But since we cannot access or change set using index we need to use a method called add()
and for more then one item the update() method.

# frist create a set
a = {1, 2}

# now lets add a item in this set. For example 3
a.add(3) # this will give {1, 2, 3}

# now lets add more then one item
# Remember when adding more then one item they must be in a list
a.add([4, 5]) # this is give {1, 2, 3, 4, 5}

How to remove an item from a set

To remove an item from a set we have to use the remove() or discard() method.

Just when using them you should know there difference which is the remove() method will raise an
error if an item given to be remove is not in the set. While on the other hand then discard() method
will not raise an error and will leave the set unchanged.

# this is how remove works
a = {1, 2, 3}

a.remove(1) # output: {2, 3}

# here it will give an error as 4 is not in this set
a.remove(4)

b = {1, 2, 3}

b.discard(1) # output: {2, 3}

# this will infact not give an error even if there is no item 4 in this set
b.discard(4)

Operators for set

We can also use diffrent operators for sets.

The & operator

If you want to find the common value in two set we can use the & operator

a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# output will also be in set {3, 4}
print(a & b)

The - operator

If you want to find the uncommon values in a which are not in b then we use the - operator, which will be a-b.
Same way if b-a then you will get then uncommon values in b which are not in a.

a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# find the uncommon values in a
# Output: {1, 2}
print(a-b)

# find the uncommon values in b
# Output: {5, 6}
print(b-a)
