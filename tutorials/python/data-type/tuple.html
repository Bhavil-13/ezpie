tuples in python

In this article you will learn about another data type like set called tuple.

tuples are like list but more closer to set as they are unchangeable but ordered.

They are written inside round brackets ( ). But these round brackets are optional,
But it is a good way of making your code look clear and at the same time simple to understand.

# This is same as a = 12, 2, 15
a = (12, 2, 15)

How to access an item in a tuple

As I told you tuple are ordered that means we can access an item in a tuple using index
for which we use the square brackets [ ]

Take an example like a tuple with 5 item that means the first item starts with 0 and ends with 4(recall range).
And if you try to get an item off range means samthing like 5 or 7 or so on will give an error called IndexError

And of course the index should and must be an integer so no to float and string.

a = (22, 19, 12, 14)

# now to access a value maybe like 19
a[1] # output: 19. Remember range again

This in fact is also called positive indexing as we are using positive numbers like 1, 0, 4, 2.
But what if you want to do it the other way round? This is then called negitive indexing.

Negitive indexing

# the same tuple a = (22, 19, 12, 14)

# now to access it in negitive indexing
a[-1] # output: 14

Slicing

To access a range of item on a tuple we use the slicing operator the colon :

a = (1, 11, 12, 13, 19, 16)

# lets try to get the 2nd to the 6th
a[1:5] # output: (11, 12, 13, 19, 16)

How to change a tuple

Just like sets tuple are unchangeable. But we can change its values when its a list
This can be done by type casting, change the tuple into a list then add or change values
and again type cast it back to a tuple.

How to unpack a tuple

When we create a tuple we assign values to it this is called 'packing' a tuple.
But what if you want to place all the items in a tuple in Variables? Then this will be called 'unpacking'.

fruits = (1, 2, 3)
(item1, item2, item3) = fruits

print(item1, item2, item3) # Output: 1 2 3

Methods for tuple

Just like other data types tuple also have some methods, which are index() and count()

Index() method

The index method is used to find the index of a value in a tuple.

For example:
if you have a tuple with values ('e', 'z', 'p', 'i', 'e'), and you want to find the index of 'i'
then we use the index() method like this:

name = ('e', 'z', 'p' , 'i', 'e')

# now to find the index of 'i'
print(name.index('i')) # output: 3

Count() method

The count() method is used to find how many same values are there in a tuple.

For example:
if you have a tuple with the same values used in the first example and you want to find how many
'e' are there. Then we use the count() method.

name = ('e', 'z', 'p' , 'i', 'e')

# now to find how many e's are there
print(name.count('e')) # output: 2
