Strings in python

In this article you will learn more about the data type string

What is string data type

A string data type is a bundle of characters(All the letters to be exact)

Strings are always at all conditions inside single or double quotation marks.
Therefore 'hello' is same as "hello", even if there are deffrent quotation marks.

And since being nothing more then simple english words string is then most easy data type.

How to create a string?

Remember double and single quotation marks, if any character or characters are inside it they are called string.

fruit = "apple"
fruit2 = 'apple'
fruit_company = """apple is good
but apple is a company"""

But if you saw that fruit_company is using triple quotation marks, is it or not that triple quotation marks
are used for multiple line Comments? than why in a variable?
Thats because its now a multi-line string! Yes you can write multiple lines in one variable using triple quotation marks.

apple
apple
apple is good
but apple is a company

String are like arrays

Don't worry if you don't known what an array is. But in simple terms an array is not a built-in data type like other data types
but instead all the sequence data types, because we can loop through only sequence data types.

Example:

Here we have a variable called companies with name of 3 companies, next we have a for loop, looping through the variable
companies then printing the value of i.

Output:

google
apple
spaceX

The same can be done using string

Here we have a variable called company with value google, next we are using a for loop to loop through the variable
company and storing value in i, and then printing the value of i.

Output:

g
o
o
g
l
e

Operations in string

Yes! we can use operators for string, but only the + and * operators.

If you join two or more strings it is called concatenation. For which we use the + operator.
And if you use the * operator it repeats the string.

Example:

>>> 'hello' + 'me'
hellome
>>> 'hello' * 3
hellohellohello

Here we are frist adding two strings 'hello and 'me', which returned hellome.
Next we are multiplying 'hello' 3 times, giving hellohellohello

Checking for letters

If you what to see if a letter is there in a string you can use the in keyword.

>>> 'p' in 'apple'
True

Or maybe the other way round

>>> 's' not in 'apple'
True

String formatting

Recap: remember we can not combine string with any other data type?

num = 12
print('a random number: ', num)

This is because string can never be combined with any other data type. Therefore convert the data type to string frist.

But! you don't always need to do that, using python format() method the same can be done!

num = 12
print('a random number: {}'.format(num))

Using the format() method we can combine a string with any data type!

Here we are frist creating a variable num with value 12, then printing a string 'a random number:', but
when using format() mathod we must place {} where we what to place other data types.

a random number: 12

Fact: When using the format() mathod we can also place a data type as per our needs.
Example:

# this is the default way
# Output: some random numbers: 1, 6 and 12
txt = 'some random numbers: {}, {} and {}'.format(1 ,6,12)

# this is the positional way
# Output: some random numbers: 12, 1 and 6
# REMEMBER index starts from 0 not 1!
txt = 'some random numbers: {2}, {0} and {1}'.format( 1 ,6,12)

Using the fromat() method we can also convert integer to binary or hexadecimal

>>> 'Binary for {0} is {0: b}'.format(10)
Binary for 10 is 1010

String methods

>>> 'ApPlE'.lower()
apple
>>> 'ApPlE'.upper()

>>> 'reactjs uses typescript'.replace('typescript' , 'javascript')
