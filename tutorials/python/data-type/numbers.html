Python number data type

There are three numeric data types in Python

  • Integer
  • Floating-point
  • Complex

Integers and floating-points are diffrent only in one thing, that is integers don't have a decimal point and floating-points
have a decimal point. Therefore 11 is an integer while 11.0 is a float.

Complex in the other hand are those with two parts, maining 1+5j where 1 is the real part and 5 is the imaginary part.

We can use the type() function to print the type of a variable. But when using type() you must keep it in the print()
function or in a variable and then print it.

Integer = 5

Complex = 1+5j
