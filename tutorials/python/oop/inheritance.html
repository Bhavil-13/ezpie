Python inheritance

This article you will get to know about inheritance.

what is inheritance in python

Inheritance is a very usefull feature in OOP.

It means that there are two or more classes with one class taken as the base(or parent) class. And rest are the inherits
from the base class, and are called derived (or child) class.

python inheritance example

Here's an example to explain it.

This may look a bit dumb to do, as we could just have created an object called tom and did this tom = Person('Tom', 'Cat')
But why this long? Well if wanted to have a diffrent method to get the favorite thing of tom and a diffrent method for asking
the enemy of jerry, will you do it on the same base class? No, because it will make no sence to create two diffrent methods
for just two objects.

Yes! Now you are getting into real programming! This is infact really big, but not much! (Your softwares are even bigger!)

Here we have a base(Person) class and two classes(Tom, Jerry) inherits from the base. Did you notice that we have created
one diffrent method in each of the child class? Well each of these methods are only accessible for it's object only.

And also because of this our classes are well in order. Which is why our base class will be short and simple.

python override method

So far we had the child class have all the attributes of the base class. But what if you want to change some of it?
This is called attributes overriding.

Try to create the __init__ method in any of the child class, try doing it in Tom and just type pass inside the method.
When you will run the code you will get an error saying, 'TypeError: __init__() takes 1 positional argument but 3 were given'

Here what we have done is, we created a same method which was in the base class. So when you run the code python
takes that method a diffrent method from the base class one, therefore instead of 3 arguments we need to give only 1.
Which is the object itself.

But if we want that method to be in the child class then we need to call it.

Notice I have used a function called super(). This function can be replaced by the name of the base class. But in case of
multiple inheritance it would be better to use the super() function as it will inherit all the attributes from all the base classes
