Multiple inheritance

In this article you will get to know how to use multiple inheritance

What is multiple inheritance?

A class in python can be derived from multiple classes. This is called multiple inheritance.

Like single inheritance, all the attributess in the base classes are inherit to the derived class.

Here's how it looks

Take this like a family tree. Where you have your parents at the top and you at the bottom.

Here if the two base classes have any attributes that attributes is also available for the derived class.

Multilevel Inheritance in python

We can, infact, inherit a class from a inheritance of another class.

In simple words a class can inherit from a class which is inherit from another class.

Here's how it will look

This can be taken as, The base class as your grandparent, the next class as your parent and the last class as you.

why multiple inheritance?

If you have though of 'why should I use multiple inheritance?' then here's why.

Take an example:
If you needed to use some classes with same attributes, but you also wanted your code a bet more formed then
you will use multiple inheritance.

This is a bit confusing. But if you focus you may understand it on your own.

Here we have a class called Bird which just takes the name of the bird and returns it. Then we have a class called
Type which tells us that what kind of bird it is. Lets go through each class.

First the Bird class. Here we are using the __init__() function to take in the name of the bird. After which it returns a
text like this 'Hi (name of bird)'.

Next is the Type class, which tells us what kind of bird it is, by it's eating habits.

Then we have a class called Eagle(You can create any bird). Which is inherit from the two classes above. Using the
super() function. And the name which is passed to the Eagle class is passed to the Bird class.
