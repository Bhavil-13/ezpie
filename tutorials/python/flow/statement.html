Python if..elif..else statement

In this article you will learn how to use if statements

If statement

In programming we need to do a lots of decision making.
If statements are a block of code to be executed when the condition is true.

Syntax

Here we have, if the condition is true then execute some code.

if..elif statement

Some times it's only if statement, but some times you need more then one posible chose.
Take an example,
You made a code in which if a user enters work it prints the work of the user, if the user enters hello then is prints 'hello',
and more. Here we need to use if..elif statement.

Syntax

Here we are trying to see if the condition is true then print True, and if the condition is false then see the next elif statement.

if..elif..else statement

Some times you may have a program which prints same result for most, therefore we use the else statement.
Take an example

You have a number between 0 and 20. You want to see what the number is by try and error.
