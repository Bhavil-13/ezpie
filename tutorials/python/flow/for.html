For loop in python

In this article you will learn what's a for loop

What's a for loop?

In python sometimes you may want to go through a list, string or other sequence type variable.
Take an example.

Here we have a list of random numbers. Next we are using the for loop to go through it.
Frist the for loop enters the list, then it gets the frist item, which is 12 and then the others.
Like this it then prints them. Did you notice that after for I have written i.

Well i in here is the variable which is storing the numbers in the variable numbers. Since
The frist number was 12 therefore i will store 12 in it, then the print function will print i.
And it's just repeated again and again till the for loop is complete looping in the variable numbers.

Output:

12
54
63
92

Using for loop to loop in a string

Yes we can use for loop to loop in a string.

Here we had a string hello. We used the for loop to loop in it and store it in variable i.
Then we are printing the variable i, which will get us output as:

h
e
l
l
o
