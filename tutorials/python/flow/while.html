While loop in python

In this article you will learn how to use and what is while loop.
Loops are used in a programming language to repeat a specific block of code.

What's a while loop?

While loop is used to run a specific block of code only if the condition is true.

It's recomended to use this loop only when you don't known the number of times you may what to run a code again.

Syntax

In a while loop first the condition is checked, if its true then the while loop will run, if not then it will be skipped.
Once the condition is true then the while loop's body is entered, where after the code is run the while loop is again checked,
this time if the condition is false then the while loop is breaked, if not then repeat till the condition is false.

Take an example:

Here we are asking the user to enter a random number in the variable num. Next we have a variable i with value 0,
whose value will change in the while loop. Next we have the while loop, where we are testing if i is not equal to num,
if it's true that i is not equal to num then we enter the body of the while loop. Here we have first we print the value of i,
then we add i with 1 and set it as its new value. then repeat the while loop.

Now when the the variable i will be more then the value of num then the condition(which is i is not equal to num) is false,
and the while loop ends. Cool right?

Else in while loop

Just like if statement while loop can also have else block.

And like I have told in the if statement article else block is executed only when the codition is not true.

Take an example:

Here we have a variable called num with value 0. Next we have the while loop which sees if the codition is true or not.
Next in the while loop body it prints the value of num and adds 1 to it as it's new value,
and repeats it till condition is false. Once the condition is false it ends the while loop and continue with the else block,
and prints 'While loop end, else block executed'.
