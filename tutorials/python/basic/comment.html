Python comments

In this article you will learn what, how, and when to use comments

What's a comment?

What are comments for starters. Comments as the word says it is a comment
they may sound unimportant to beginners as a comment is only a plain text to explain the code
but that's not true!
imagine you wrote a program to make a computer talk. Many days later when you see the code you get confused!
that's because you can't read it or understand it, here comes comments to help you remember it!

How to write a comment

to write a comment in Python and Python only, we use the hash # sign
This is only for single line comment, that means the line after the hash is a comment not the next or before line.
Since a comment is of no use for Python It ignores it and continue

# This is a single line comment
# This program prints 'Hello world' as output
print("hello world")

Multi-line comment

Some times we need to write a lots of lots of comments so we use the multi-line commenting
which starts with a ' ' ' and ends the same way. Even " " " is appropriate to use
spacing is for clear reading only. Not to be done will coding

'''This is a multi-line comment
This program prints 'Hello world' as output'''
print("hello world")
