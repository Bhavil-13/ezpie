Python Operators

In this article you will learn about operators and how to use them.

What are Operators

Operators are same as signs used in maths like the + operator, - operator, x operator and the รท operator.
These are infact not the same way typed in any programming language.

To type them in a programming language you use these signs instead, but for + and - it's the same.
For multiplction we use the * sign, for division we use the / sign.

But these are not all of them! programming languages also have the remander sign %, the floor-division sign //,
and the exponent sign **.

But These are not the only once, all programming languages have upto seven type of operators!
Arithmetic operators, Comparison operators, Logical operators, Assignment operators and bitwise operators.
These are infact the most common operators in python.

We will go through each one of them.

Arithmetic Operators

Arithmetic operators are the maths sign which I have alrady explained.
Here's an example.

num1 = 123
num2 = 432
sum = num1 + num2
diff = num1 - num2
multipul = num1 * num2
qudrent = num1 / num2

Here we have two variables with two numbers 123 and 432. To add these numbers I used the + operator,
to subtract the - operator, to multiply the * operator and to Divide the / operator.

Arithmetic operators are easy to learn so would be easy to learn the others?
will yes since python is quite like english so no joke, it's easy to learn the other operators as will.

Comparison in python

comparision operators, as it says are used to compare variables.
Take an example, you have one apple and one banana both are one right? but how to compare it in python?
will simple just use the == operator.

apple = 1
banana = 1
print(apple == banana)

Output

True

And you get a True that means apple and banana are equal. But what if they are less or more or not equal or
somthing somthing. While then we will use these operators.

apple = 3
banana = 2
print(apple > banana)
print(apple < banana)
print(apple != banana)
print(apple >= banana)
print(apple <= banana)

Output

True
False
True
True
False

Let me explain what we did, here we used the > sign(more then), then the < sign(less then),
then != sign(not equal), then >=(more then or equal to) and then <=(less then or equal to) .

This may be deficult to understand but try each sign then you may get it.

Logical Operators

Logical operator are used for when you want multiple reason for somthing to happen.
Means the and, or and not operators.

The and operator is use when both the variables are true.
The or operator is used when only one variable is true.
The not operator is used when the variable is false.

Here's an example

x = True
y = False
print(x and y)
print(x or y)
print(not x)

Output

False
True
False
