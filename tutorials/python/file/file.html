Operations in file

In this article you will get to know one of the awsome things you can do using python.

Files in python

As we know that RAM(Random Access Memory) is only one sesson use. Means that the data stored in it will be
deleted once the device is turned off. Therefore we use files to store data forever(till you delete it)

Just like how we do things to a file the same way is using python:

  1. Open the file
  2. Make edits to the file
  3. Close the file

We will go through each stage one by one.

python open file

First we need to open a file, so we can make edits to it.

Luckly, python has a build-in-function called open() to open a file. This function create's the variable, which is used,
into an object, which is then used to make edits.

file = open("file.txt")
file = open("C:\ezpie-python\file.txt")

The first one is used in case we have a file were our python file is. And the second one is used to define the exact
location of the file.

When using python for using files, we need to also define how we are going to use that file. Means that, are we going
to read from it? or are we going to write to it?

We can do this by telling what mode it is. We just need to specify what way the file will be used, read(r), write(w), or
append(add more, a). We can also tell weather it is to be in text or binary mode.

f = open("file.txt") # read in text mode
f = open("file.txt", "w") # write in text mode
f = open("img.jpg", "r+b") # read and write in binary mode

python close file

Like we close a file after using it, we also need to do the same while using python.

To close a file in python we use the close() method in python.

f = open("test.txt", "r")
# Now close the file after reading it
f.close()

NOTE TO BE NOTED: This is not a safe way as this can give us an exception, which in ture can stop the operation and
the code will stop without closing the file!

But python has a quite safer way, which is try...finally block.

Like the to words say, try to execute some code and finally end up by executing these codes.

Like this one thing is sure, that even if a exception occur then also the finally block will be executed, therefore our file
will be closed!

But still the best way is only the with statement's way!

This way it's ensure that the file is closed when the block inside the with statement is executed. We don't even need
to call the close() method. It is done by the with statement itself.

How to write to file

When writing to a file we need to mention, do we need to write to a file? Or add more text to a file?

Note: Writing to a file is deffirent from appending to a file. Writing will remove all present data in a file with newly
typed data. While appending will add the typed text to the file instand of replacing it.

More info: With write as the mode, if a file named mentioned does not exist then python will create a new file with that
name and data!

Here we simply are tring to open a non-existing file called does-not-exist.txt (The name means it). But like I said, "if a file
named mentioned does not exist then python will create a new file with that name and data"
And therefore when you will see your folder where your python file is located, there will be a file named ''does-not-exist.txt".

Next we are using a method called write, this method is used to write to a file with data typed in the round bracket. And
you may have notice the \n, this is just a simple way of saying next line or new line. Since python will not understand where
it needs to give a new line we just need to type it in.

Reading a file in python

To read a file in python we must set the mode to r(read)

With this mode we use the read method which can take argument or be empty. With no argument the method will return
the whole data stored in the file. But with argument it will return the number of characters.

Take for example, if you pass the argument as 4 then it will return the first 4 characters in the file(counting spaces)
