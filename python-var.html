How to name a variable

do and don't to name a variable in python:

  • Variable name must not start with a number(1name, 23name)
  • Variable name must start with latters or underscore(name, _name)
  • Variable name can contain number but after a latter(name1, n1ame)
  • Variable name must not have space in between(my age)

# Legal ways
name = 'Tom'
name_first = 'Tom'
name1 = 'Tom'
# Illegal ways
1name = 'Tom'
name-first = 'Tom'
name first = 'Tom'
