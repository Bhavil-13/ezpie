Break keyword in python

In this article you will learn what is the break keyword.

Just like the continue keyword the break keyword is used to alter a loop. But using continue will only
skip one specific condition, while the break keyword will end the whole loop.

Try going through this flowchat and you will understand the maining of both the keywords.

What's the break keyword?

The break keyword is used the break the loop when a condition is true.

If you use a nested loop(a loop inside a loop) the break keyword will only end the loop it is inside.
Like if a for loop(loop 2) is in a for loop(loop 1), and the break keyword is inside the second for loop(loop 2),
then the second loop will end(loop 2 not loop 1).
