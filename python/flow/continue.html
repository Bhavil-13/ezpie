continue Keyword in python

In this article you will learn what's a continue keyword in python

Needs of a continue keyword

In any programming language there are keywords to alter a loop. In python they are 'continue' and 'break'.

Sometimes you may want to end your loop in middle of the execution of code. Therefore we use 'continue'
and 'break'.

Using continue keyword

The continue keyword is used to skip the rest of the code below it. The continue keyword does not finish the loop
it only skips the left body of the loop.
