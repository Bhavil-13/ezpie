Python Input And Output

In this article, you will learn about one vary importent user interface called input output(As I like to call)

User interface is an importent topic as most apps, softweres and websites wants there user to have
a good experience. Therefore understanding inputs and outputs are importent to known.
But these user interface are diffrent from those which you may have seen.

Output in python

Till now you may have seen that we had our output using print() function.
And it is the only option in python to get an output.

print('Just like this')

Output

'Just like this'

This example may just change what you may have thougth about the print function.

num = 120
print('The number is:', num)

Output

'The number is 120'

You may have notice that we have put space between the two statement using a comma.
And in fact you can also use the + oprent. This is explained in a diffrent article.

num = 120
print('The number is: '+num)

Output

'The number is 120'

You may have notice the lettle gap in the string inside the print function print('The number is: ')
after the colon, that's because when using the + oprent it adds the two.
Giving 'The number is120', which we don't want.
