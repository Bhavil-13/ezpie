Python Data Type

In this article you will learn What are data types in python

when it comes to programming data type is an importent concept.
As the variables we define each have a data type.
As of Python it has the following data types:

Text type: string

Numerical type: integer, floating-point, complex

Boolean type: bool

Sequence type: list, range, truple

Mapping type: dict

None type: NoneType

Binary type: memoryview, bytearray, bytes

Set type: frozenset, set

We will go through each one but mainly those which are mainly used.

Integer

As you saw in above The Numerical Type in python are three. Integer, floating-point and complex.
Integers are same as integers in maths, but Integers in python are only whole numbers, not
dacimal points.

# This is a variable with data type integer
number = 1230
# but this is not at all a variable with data type integer
number2 = 1230.123

If you want to known the data type using python, you can use the type() method.

number = 342
print(type(number))
number2 = 342.34
print(type(number2))

If you run it you will get the output as:

<class 'int'>
<class 'float'>

NOTE: Did you saw that the two variable number and number2 have same numbers but one is a whole
number and the other is a decimal. But still diffrent, you will see that later.

A quick think before you go for floating-point, Integers can take any length of number, as per memory.
As pre me floating-points can take upto 15 decimal places, and as pre google its 15 to 17

Floating-point

Floating-point are data-type with decimal points just like I told while explaining integer.
Since I have already explained we will just jump to the main part.

floating_Number = 1243.431
print(type(floating_Number))

After running it we will get the output as:

<class 'float'>

Complex

As of integer and floating-point they are in programming called real numbers.
but Python and other programming language they can also take complex number.
complex numbers are those which have one real number and one imaginary number, a+bj
here a is the real part while b is the imaginary part.

num = 3+5j
print(type(num))

after running this, you will get the output as:

<class 'complex'>

Boolean

Boolean is the data type with only two values which are True and False.
You may have seen that I the true and false have there first latter capital, that's because
true and false are the only keywords with capital first latters.

bool = True
print(type(bool))

When you will running it you will get the output as:

<class 'bool'>

Quick thing by now you may think that learning boolean data type is wast of time but no its not,
boolean data type are mostly used in databases so when you may use database in python
you may use boolean more often.

Range

Range is a function which will always return a sequence of numbers
starting from 1 by defult but can be changed, to a number given by you as the user,
by 1 step by defult but again can be changed

numbers = range(1, 10, 1)
print(numbers)

Here we have define a variable called numbers with a value of range(1, 10, 1).
May seem a bit to think but there is nothing to think,
cause the first 1 here is the starting point, 10 is the ending point and the last 1 is the steps to be taken.

After running this you may think that it will print 1 to 10, but try once then see.

Truples can have any data type in them just like list

Dictionary

Dictionary in programming are vary useful as they can store data in a variable with sub-variables.
Here's an example to tell what i mean.

my_dict = { 'company': 'Apple', 'owner': 'Tim cook' }
print(my_dict)

Here we have define a variable called my_dict, with the sub-variables company and owner,
with values Apple and Tim cook respectively. And then prints the variable.

But this will only print the dictionary which is:

{'company': 'Apple', 'owner': 'Tim cook'}

But we want the values and the good thing is we need no for loop for it.
We can use the sub-variables using variable name[ property ]. where variable name will be
my_dict and the property will be company or owner.

my_dict = { 'company': 'Apple', 'owner': 'Tim cook' }
print(my_dict[company])
print(my_dict[owner])

After running it you will get what you want.

'Apple'
'Tim cook'
