Python Variables

In this article you will learn what's a variable

What's a variable?

A variable can be taken as a box in which you have a thing for example a photo of a house
or an old item or anthing

How to declare a variable

To declare a variable we don't have to do anything spacial (if you know any other programming language) we just
have to gave the variable a name and a value, simple as that.
and a variable can also be changed. for example:

# This is a variable called random with a value 20
random = 20

Here we have declared a variable called random with a value of 20.
Take it in the simple way random is box 20 is photo.
So in that manner 20 can be replace right?
will yes! because 20 is photo and that photo can be replace with cloth.

random = 20
random = 413

random is box 20 is photo and now random(box) cantains 413(cloth)!

case-sensitive

quick thing did you thought till now that Apple is same as apple or any??
well you will be suprise to know Apple is not the same as apple or so!?!?
that's because python and all other programming languages are case-sensitive meaning lower-case is
diffrent from upper-case. example:

Apple = 2

Here Apple will not change to 15 as Apple is deffrent from apple

How to name a variable

do and don't to name a variable in python:

  • Variable name must not start with a number(1name, 23name)
  • Variable name must start with latters or underscore(name, _name)
  • Variable name can contain number but after a latter(name1, n1ame)
  • Variable name must not have space in between(my age)

# Legal ways
name = 'Tom'
name_first = 'Tom'
name1 = 'Tom'

# Illegal ways
1name = 'Tom'
name-first = 'Tom'
name first = 'Tom'
