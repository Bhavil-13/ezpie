Python type Conversion

In this article you will learn what and where to use type conversion.

But you must known about python data type first.

What is data type conversion

What is data type conversion you may be thinking. Well data types I have alrady explained,
but some times you may want to change the data type of a variable.

Take an example, you previously learn how to ask the user to give an input.
So you may have tred to make a program to take two inputs and do some maths on it like add or subtract.
But you may have got some funny result like,
the first number given is 12, and the second is 30, so if added answer will be 42 right?
but what you got 1230? after printing it.
What happen? will all inputs are in string data type!
therefore we need to convert it to interger or floating-point.

Types of type conversion

But that's not it there are two types of data type conversion.
Implicit Type Conversion
Explicit Type Conversion

Implicit type conversion

implicit type conversion is when python auto converts the data type.

# These are two variables with diffrent data type
num1 = 123
num2 = 431.41
sum = num1 + num2
print(type(sum))

Output

<class 'float'>

Here we added two numbers, with diffrent data type integer and floating-point.
then printed the data type of sum which give class as float.

Explicit type conversion

Now lets add a string and an integer.

num1 = 22
num2 = '22'
sum = num1 + num2
print(type(sum))

Output

Traceback (most recent call last): File "c:\python\main.py", line 3, in <module>
sum = num1 + num2
TypeError: unsupported operand type(s) for +: 'int' and 'str'

Here we have a TypeError, which means That string can not be added to number types.
Therefore we need to convert the num2 to an integer first.

num1 = 22
num2 = int('22')
sum = num1 + num2
print(type(sum))

Output

<class 'int'>

Here we had a string '22' which we then converted into an integer using the int function.

By just typing the keyword of the data type you can convert any data type to the other.
